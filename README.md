# LocTrackerSabre

## Opis projektu
1. Okresowe generowanie informacji o lokalizacji z wielu urządzeń (symulacja z aplikacji backendowej - "Gate").
2. Poprzez pub/sub przesyłanie informacji o lokalizacji do innej aplikacji backendowej - "Location".
3. Wyświetlanie informacji o lokalizacji w aplikacji frontendowej "Web" wraz z możliwością wyświetlenia informacji historycznych.
4. Przechowywanie informacji o lokalizacji w Firestore.
