# LocTrackerSabre

### English below

## Opis projektu
1. Okresowe generowanie informacji o lokalizacji z wielu urządzeń (symulacja z aplikacji backendowej - "Gate").
2. Poprzez pub/sub przesyłanie informacji o lokalizacji do innej aplikacji backendowej - "Location".
3. Wyświetlanie informacji o lokalizacji w aplikacji frontendowej "Web" wraz z możliwością wyświetlenia informacji historycznych.
4. Przechowywanie informacji o lokalizacji w bazie danych Firestore.

### English

## Project description

1. Periodic generation of location from multiple devices (simulated by the backend application “Gate”).
2. Sending location information to another backend application called “Location” via pub/sub.
3. Displaying location information in the frontend application called “Web”, with the ability to view historical data.
4. Storing location information in Firestore database.
