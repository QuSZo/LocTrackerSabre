# LocTrackerSabre

## Opis projektu
1. Okresowe generowanie informacji o lokalizacji z wielu urządzeń (symulacja z aplikacji backendowej).
2. Poprzez pub/sub przesyłanie informacji o lokalizacji do innej aplikacji backendowej.
3. Wyświetlanie informacji o lokalizacji w aplikacji frontendowej (ewentualnie możliwość zaznaczenia obszaru zabronionego).
4. Przechowywanie informacji o lokalizacji w Cloud Storage.
5. Poprzez pub/sub wysyłanie maila z alertem o tym, że urządzenie znajduje się w zakazanej lokalizacji.
